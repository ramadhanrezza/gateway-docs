Getting Started
###############

Prerequisites
*************

Before starting, you need to do these step for this service to work. 

* composer

 .. code:: sh

	curl -sS https://getcomposer.org/installer -o composer-setup.php

	sudo php composer-setup.php

	composer

* git

 .. code:: sh

    sudo apt install git

* redis

 .. code:: sh

	sudo apt install redis-server
